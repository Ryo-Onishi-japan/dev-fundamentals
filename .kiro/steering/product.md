# Product Steering

## Purpose

This project is a hands-on learning repository designed to teach foundational developer skills. The primary goal is to provide a practical environment for mastering GitHub workflows, Java programming, PL/SQL database interactions, and AI-driven coding practices.

## Core Capabilities

- **Skill Development:** Offers a structured path for learning essential software development skills.
- **Tutorial Project:** Features an "employee management system" as a practical application for the concepts being taught.
- **AI-Assisted Development:** Integrates with Gemini CLI and follows a Kiro-style Spec-Driven Development process to guide AI-assisted coding.

## Target Audience

- Developers new to the workforce.
- Students learning software development.
- Anyone looking to practice and improve their skills in the specified technologies.
